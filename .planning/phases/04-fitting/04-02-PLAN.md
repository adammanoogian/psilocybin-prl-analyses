**batch.py — fit_batch() function**

Create `src/prl_hgf/fitting/batch.py` with:

```
def fit_batch(
    sim_df: pd.DataFrame,
    model_name: str = "hgf_2level",
    n_chains: int = 4,
    n_draws: int = 1000,
    n_tune: int = 1000,
    target_accept: float = 0.9,
    random_seed: int = 42,
    cores: int = 1,
    output_path: Path | None = None,
) -> pd.DataFrame:
```

Implementation:
- Accept sim_df (the tidy trial-level DataFrame from simulate_batch, with
  columns: participant_id, group, session, cue_chosen, reward, etc.)
- Group by (participant_id, group, session) to get unique participant-sessions
- For each participant-session:
  1. Extract the trial data subset
  2. Build input_data_arr (n_trials, 3) and observed_arr (n_trials, 3) from
     cue_chosen and reward columns using the same partial-feedback logic as
     prepare_input_data: input_data[t, choice] = reward, observed[t, choice] = 1
  3. Extract choices_arr = cue_chosen values as int array
  4. Call fit_participant() with appropriate arguments
  5. Collect summary_rows and flagged status
  6. Add "flagged" column to each row (bool)
- Aggregate all summary_rows into a single DataFrame
- Log progress: every N participant-sessions, print elapsed time and count
- If output_path provided, save results DataFrame as CSV
- Return the results DataFrame

The results DataFrame columns MUST match FIT-04 spec:
participant_id, group, session, model, parameter, mean, sd, hdi_3%, hdi_97%,
r_hat, ess, flagged

JIT pre-warm: Before the main loop, call the Op once with dummy values to
trigger JAX JIT compilation (same pattern as simulation/batch.py).

Error handling: If a single fit raises an exception (e.g., SamplingError from
divergences), catch it, log a warning with participant_id, and add rows with
NaN values for that participant. Do not crash the entire batch.

- Support command-line argument for model_name (default: "hgf_2level")

The script should fit ONE model variant at a time (run it twice for both
models). This is simpler and allows the user to start with 2-level only.
