**Grid decode module** — Create `src/prl_hgf/power/grid.py`:

```python
def decode_task_id(
    task_id: int,
    n_per_group_grid: list[int],
    effect_size_grid: list[float],
    n_iterations: int,
) -> tuple[int, float, int]:
```

Logic (from Research Pattern 1):
- `n_d = len(effect_size_grid) * n_iterations`
- `n_idx = task_id // n_d`
- `d_idx = (task_id // n_iterations) % len(effect_size_grid)`
- `iter_idx = task_id % n_iterations`
- Return `(n_per_group_grid[n_idx], effect_size_grid[d_idx], iter_idx)`
- Raise `IndexError` with descriptive message if `task_id >= total_grid_size`.

Also add `total_grid_size(n_per_group_grid, effect_size_grid, n_iterations) -> int` that returns `len(n_per_group_grid) * len(effect_size_grid) * n_iterations`.

Use `from __future__ import annotations`, NumPy-style docstrings, type hints.
