**3. src/prl_hgf/gui/explorer.py:** Create the `ParamExplorer` class. This is
the main deliverable. Architecture details:

**Constants:**
- `GUI_SEED = 0` — fixed seed for warm-up simulation reproducibility.
- `_HGF_PARAMS = frozenset({"omega_2", "omega_3", "kappa", "mu_1_0", "mu_3_0"})` — params that require a forward pass.
- `_RESPONSE_PARAMS = frozenset({"beta", "zeta"})` — params that only need softmax recompute.
- `PROFILES` dict with three presets (healthy_baseline, post_concussion, post_psilocybin) using values from 07-RESEARCH.md.

**ParamExplorer.__init__(self, config=None):**
- Load config via `load_config()` if not provided.
- Generate trial sequence: `trials = generate_session(config, seed=GUI_SEED)`.
- Extract ground-truth probabilities: `gt_probs = np.array([t.cue_probs for t in trials])` shape (n_trials, 3).
- Extract phase boundaries for shading (stable vs volatile).
- Warm up JAX: print "Warming up JAX (one-time ~5-10s)..." then run `simulate_agent` once with default params to JIT-compile both the trial-by-trial and batch forward pass paths. Use the warm-up result to create fixed `_inp_data_` and `_obs_mask_` via `prepare_input_data(trials, result.choices, result.rewards)`.
- Also run one batch `net.input_data()` call to JIT-compile the batch path.
- Create sliders using `ipywidgets.FloatSlider` with `continuous_update=False`:
  - omega_2: min/max from `PARAM_BOUNDS["omega_2"]`, step=0.1, default=-3.0
  - omega_3: min/max from `PARAM_BOUNDS["omega_3"]`, step=0.1, default=-6.0
  - kappa: min/max from `PARAM_BOUNDS["kappa"]`, step=0.05, default=1.0
  - beta: min/max from `PARAM_BOUNDS["beta"]`, step=0.1, default=2.5
  - zeta: min/max from `PARAM_BOUNDS["zeta"]`, step=0.1, default=0.2
  - mu_1_0: min=0.0, max=1.0, step=0.05, default=0.5 (initial expected_mean for binary input nodes)
  - mu_3_0: min=-4.0, max=4.0, step=0.1, default=0.0 (initial mean for volatility node)
  Store sliders in `self._sliders` dict keyed by param name.
- Create model toggle: `ipywidgets.ToggleButtons(options=["2-level", "3-level"], value="3-level")`.
- Create preset buttons: `ipywidgets.ToggleButtons(options=list(PROFILES.keys()), value=None)` — allow None so no preset is active by default.
- Create the 4-panel matplotlib figure (DO NOT call plt.show()):
  - axes[0]: "Belief trajectories P(reward)" — 3 belief lines + 3 ground-truth dashed lines, ylim (0,1)
  - axes[1]: "Volatility mu2" — 1 line for shared volatility (3-level only)
  - axes[2]: "Choice probabilities P(choose cue)" — 3 lines, ylim (0,1)
  - axes[3]: "Effective learning rate" — 3 lines (one per cue), ylim (0,1)
  - All share x-axis (trial number). Add phase boundary shading (axvspan with alpha=0.05, alternating colors for stable/volatile).
  - Store line objects as `self._belief_lines`, `self._gt_lines`, `self._mu2_line`, `self._prob_lines`, `self._lr_lines`.
- Observe all sliders: `slider.observe(self._on_slider_change, names="value")`.
- Observe model toggle: `model_toggle.observe(self._on_model_toggle, names="value")`.
- Observe preset toggle: `preset_toggle.observe(self._load_preset, names="value")`.
- Run initial forward pass + plot update so the figure is populated on creation.

**ParamExplorer._run_forward_pass(self):**
- Read current model type from toggle.
- If 2-level: `net = build_2level_network(omega_2=slider_val)`.
- If 3-level: `net = build_3level_network(omega_2=..., omega_3=..., kappa=...)`.
- Set initial beliefs on INPUT_NODES: for each idx in INPUT_NODES, set `net.attributes[idx]["expected_mean"]` to mu_1_0 slider value. For 3-level, also set `net.attributes[VOLATILITY_NODE]["mean"]` to mu_3_0 value.
- Run batch forward pass: `net.input_data(input_data=self._inp_data_, observed=self._obs_mask_)`.
- Extract beliefs: use `extract_beliefs(net)` for 2-level, `extract_beliefs_3level(net)` for 3-level.
- Cache beliefs in `self._cached_beliefs_`.
- Extract effective learning rate per cue: `traj[node]["expected_precision"] / traj[node]["precision"]` for BELIEF_NODES (1, 3, 5). Cache in `self._cached_lr_`.

**ParamExplorer._update_choice_probs(self):**
- Read beta and zeta from sliders.
- Stack mu1 from cached beliefs: `np.column_stack([beliefs["p_reward_cue0"], ..., beliefs["p_reward_cue2"]])` shape (n_trials, 3).
- Compute stickiness array from the warm-up choices (`self._warmup_choices_`).
- Compute logits = beta * mu1 + zeta * stick.
- Use `scipy.special.softmax(logits, axis=1)` to get choice probabilities.
- Store in `self._cached_choice_probs_`.

**ParamExplorer._update_plot(self):**
- For each cue i (0,1,2): `self._belief_lines[i].set_ydata(beliefs[f"p_reward_cue{i}"])`.
- Ground-truth lines: `self._gt_lines[i].set_ydata(self._gt_probs[:, i])` (these are static, set once).
- Volatility line: if 3-level and "mu2_volatility" in beliefs, `self._mu2_line.set_ydata(beliefs["mu2_volatility"])`. Call `self._axes[1].relim()` + `self._axes[1].autoscale_view()` since mu2 range varies.
- Choice prob lines: `self._prob_lines[i].set_ydata(self._cached_choice_probs_[:, i])`.
- Learning rate lines: `self._lr_lines[i].set_ydata(self._cached_lr_[:, i])`.
- Call `self._fig.canvas.draw_idle()`.

**ParamExplorer._on_slider_change(self, change):**
- Determine which parameter changed from `change["owner"].description`.
- If param is in `_HGF_PARAMS`: call `_run_forward_pass()`.
- Always call `_update_choice_probs()` and `_update_plot()`.

**ParamExplorer._on_model_toggle(self, change):**
- `is_3level = (change["new"] == "3-level")`.
- Show/hide axes[1] (volatility panel): `self._axes[1].set_visible(is_3level)`.
- Show/hide 3-level-only sliders (omega_3, kappa, mu_3_0): `slider.layout.display = "" if is_3level else "none"`.
- Call `self._fig.tight_layout()` + `self._run_forward_pass()` + `_update_choice_probs()` + `_update_plot()`.

**ParamExplorer._load_preset(self, change):**
- Get profile dict from `PROFILES[change["new"]]`.
- Call `_unobserve_all()` to suppress intermediate callbacks.
- Set each slider value from the profile dict.
- Call `_observe_all()` to re-enable callbacks.
- Call `_run_forward_pass()` + `_update_choice_probs()` + `_update_plot()`.

**ParamExplorer._unobserve_all(self) / _observe_all(self):**
- Loop through `self._sliders.values()` and call `unobserve/observe` with `_on_slider_change` and `names="value"`.

**ParamExplorer.display(self):**
- Assemble layout with `ipywidgets.VBox` and `ipywidgets.HBox`:
  - Top row: model toggle + preset buttons
  - Left column: all sliders stacked vertically
  - Right: the figure (via `self._fig.canvas`)
- Return the assembled widget (do NOT call `display()` — let the notebook cell do that).

Use `from __future__ import annotations` at top. NumPy-style docstrings on all public methods. Type hints throughout.
