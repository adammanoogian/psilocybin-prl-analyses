```python
"""PAT-RL binary-state trial sequence generator (parallel stack).

Generates the 192-trial / 4-run structure described in
:mod:`prl_hgf.env.pat_rl_config`: hazard-driven binary state reversals,
uniform-random 2x2 reward/shock magnitudes, Delta-HR stub (N(-3,3) on
dangerous, N(0,3) on safe), and cumulative outcome timing in seconds.

This module is independent of :mod:`prl_hgf.env.simulator`; the PAT-RL
dataclass :class:`PATRLTrial` does not inherit from / share fields with
:class:`prl_hgf.env.simulator.Trial`.
"""

from __future__ import annotations

from dataclasses import dataclass
```

```python
@dataclass(frozen=True)
class PATRLTrial:
    """One PAT-RL trial record.

    Parameters
    ----------
    trial_idx : int
        Session-level trial index 0..n_trials-1.
    run_idx : int
        Run index 0..n_runs-1.
    trial_in_run : int
        Trial index within this run, 0..trials_per_run-1.
    regime : str
        "stable" or "volatile" from PATRLTaskConfig.run_order[run_idx].
    state : int
        Binary context state: 0 = safe, 1 = dangerous.
    reward_mag : float
        Magnitude if outcome == reward (one of task.magnitudes.reward_levels).
    shock_mag : float
        Magnitude if outcome == shock (one of task.magnitudes.shock_levels).
    delta_hr : float
        Per-trial Delta-HR covariate (anticipatory bradycardia, bpm).
    outcome_time_s : float
        Cumulative time of outcome onset within the session, in seconds.
    """
    trial_idx: int
    run_idx: int
    trial_in_run: int
    regime: str
    state: int
    reward_mag: float
    shock_mag: float
    delta_hr: float
    outcome_time_s: float
```

Implement helpers (all pure, all accept an `rng: np.random.Generator` that the
caller seeded):

1. `generate_state_sequence(n_trials: int, hazard: float, initial_state: int, rng) -> np.ndarray`
   - For each trial `t >= 1`: draw `flip = rng.random() < hazard`;
     `state[t] = 1 - state[t-1]` if flip else `state[t-1]`.
   - `state[0] = initial_state`.
   - Return int np.ndarray of shape `(n_trials,)`.

2. `generate_magnitudes(n_trials: int, reward_levels: tuple[float, ...], shock_levels: tuple[float, ...], rng) -> tuple[np.ndarray, np.ndarray]`
   - Draw 4-cell index uniformly per trial: `cell = rng.integers(0, 4, size=n_trials)`.
   - Map cell `-> (reward_levels[cell // 2], shock_levels[cell % 2])` (or any
     stable bijection — the property test only checks all 4 combinations occur).
   - Return `(reward_mag_arr, shock_mag_arr)` both float64.

3. `generate_delta_hr_stub(state_seq: np.ndarray, stub_config, rng) -> np.ndarray`
   - For each trial, pick distribution based on state (`0=safe`, `1=dangerous`).
   - Sample from `rng.normal(mean, sd)` and clip to `stub_config.bounds`.
   - Return `(n_trials,)` float64 array.

4. `compute_outcome_times_s(n_runs: int, trials_per_run: int, timing: TimingConfig, run_gap_s: float = 15.0) -> np.ndarray`
   - Within-run trial onset = `trial_in_run * timing.trial_duration_s`.
   - Outcome onset within trial = `cue_duration_s + anticipation_s`.
   - Add cumulative offset per run = `run_idx * (trials_per_run * timing.trial_duration_s + run_gap_s)`.
   - Return `(n_trials,)` float64 array of absolute outcome times in seconds.

All four functions live at module scope. Use NumPy-style docstrings and
native type hints.

Add the top-level generator to `pat_rl_sequence.py`:

```python
def generate_session_patrl(
    config: PATRLConfig,
    seed: int,
    delta_hr_override: np.ndarray | None = None,
) -> list[PATRLTrial]:
    """Generate a full PAT-RL session under `config`.

    Parameters
    ----------
    config : PATRLConfig
        Loaded PAT-RL configuration.
    seed : int
        Master seed; each subroutine spawns its own child stream via
        SeedSequence(seed).spawn(4) -> (state, magnitudes, delta_hr, reserved).
    delta_hr_override : np.ndarray or None, optional
        If provided (shape ``(n_trials,)``), bypass the stub and use the
        supplied values.  Values are still clipped to
        ``config.task.delta_hr_stub.bounds``.

    Returns
    -------
    list[PATRLTrial]
        Length ``config.task.n_trials``.
    """
```

Algorithm:
1. `ss = np.random.SeedSequence(seed)`;
   `rng_state, rng_mag, rng_dhr, _reserved = [np.random.default_rng(s) for s in ss.spawn(4)]`.
2. For each `run_idx` in `range(config.task.n_runs)`:
   - Pick `regime = config.task.run_order[run_idx]`;
     `hazard = config.task.hazards.stable if regime == "stable" else config.task.hazards.volatile`.
   - Initial state: first run = `0` (safe); subsequent runs preserve the last
     state of the previous run.
   - `state_run = generate_state_sequence(trials_per_run, hazard, initial_state, rng_state)`.
   - Append to session `state_full`.
3. `reward_mag_arr, shock_mag_arr = generate_magnitudes(n_trials, reward_levels, shock_levels, rng_mag)`.
4. If `delta_hr_override is not None`: `delta_hr_arr = np.clip(override, lo, hi)`.
   Else: `delta_hr_arr = generate_delta_hr_stub(state_full, config.task.delta_hr_stub, rng_dhr)`.
5. `outcome_time_arr = compute_outcome_times_s(n_runs, trials_per_run, config.task.timing)`.
6. Zip into a list of `PATRLTrial` records:

```python
trials: list[PATRLTrial] = []
for trial_idx in range(config.task.n_trials):
    run_idx = trial_idx // config.task.trials_per_run
    trial_in_run = trial_idx % config.task.trials_per_run
    trials.append(PATRLTrial(
        trial_idx=trial_idx,
        run_idx=run_idx,
        trial_in_run=trial_in_run,
        regime=config.task.run_order[run_idx],
        state=int(state_full[trial_idx]),
        reward_mag=float(reward_mag_arr[trial_idx]),
        shock_mag=float(shock_mag_arr[trial_idx]),
        delta_hr=float(delta_hr_arr[trial_idx]),
        outcome_time_s=float(outcome_time_arr[trial_idx]),
    ))
return trials
```
