<task type="auto">
  <name>Task 1: Create power/contrasts.py with JZS BF and contrast functions</name>
  <files>src/prl_hgf/power/contrasts.py</files>
  <action>
Create `src/prl_hgf/power/contrasts.py` with these functions:

1. `compute_jzs_bf(did_psi: np.ndarray, did_plc: np.ndarray, r: float = math.sqrt(2) / 2) -> float`
   - Takes two 1-D arrays of per-participant DiD values (psilocybin group, placebo group).
   - If either array has len < 2, return float("nan").
   - Compute Welch t-test: `t_val, _ = stats.ttest_ind(did_psi, did_plc, equal_var=False)`.
   - Call `pg.bayesfactor_ttest(t=t_val, nx=len(did_psi), ny=len(did_plc), paired=False, alternative="two-sided", r=r)`.
   - Return `float(bf10)`.

2. `compute_did_contrast(fit_df: pd.DataFrame, parameter: str, session_a: str, session_b: str) -> tuple[np.ndarray, np.ndarray]`
   - Filters fit_df to the given parameter.
   - Pivots to wide form: index=[participant_id, group], columns=session, values=mean.
   - Computes DiD = session_a - session_b per participant.
   - Returns (psi_did, plc_did) as two numpy arrays.
   - Raises ValueError if parameter not found in fit_df.

3. `compute_linear_trend_contrast(fit_df: pd.DataFrame, parameter: str) -> tuple[np.ndarray, np.ndarray]`
   - Filters fit_df to the given parameter.
   - Pivots to wide form with sessions as columns.
   - Computes trend = -1 * baseline + 0 * post_dose + 1 * followup per participant.
   - Returns (psi_trend, plc_trend) as two numpy arrays.

4. `compute_all_contrasts(fit_df: pd.DataFrame, parameter: str = "omega_2", bf_threshold: float = 10.0) -> list[dict]`
   - Calls compute_did_contrast + compute_jzs_bf for three contrast types:
     (a) did_postdose: session_a="post_dose", session_b="baseline"
     (b) did_followup: session_a="followup", session_b="baseline"
     (c) linear_trend: uses compute_linear_trend_contrast
   - Returns a list of 3 dicts, each with keys: sweep_type (str), bf_value (float), bf_exceeds (bool).
   - sweep_type values: "did_postdose", "did_followup", "linear_trend".

Use `from __future__ import annotations` at top. NumPy-style docstrings. Type hints with 3.10+ syntax. Import math, numpy, pandas, scipy.stats, pingouin.

IMPORTANT: `alternative="two-sided"` always (pingouin only supports two-sided JZS BF). Default `r=math.sqrt(2) / 2` matches JASP default.
