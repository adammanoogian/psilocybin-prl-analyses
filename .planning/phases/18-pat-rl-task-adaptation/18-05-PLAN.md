in a future phase (dcm_pytorch v0.4+). Phase 18 ships the producer side
only.

Primary function:

```python
def export_subject_trajectories(
    participant_id: str,
    idata: az.InferenceData,
    trials: list[PATRLTrial],
    choices: np.ndarray,          # (n_trials,) int {0,1} recorded choices
    model_name: str,              # "hgf_2level_patrl" | "hgf_3level_patrl"
    output_dir: Path,
) -> Path:
    """Export one subject's per-trial HGF trajectory CSV.

    Runs a single forward pass of the PAT-RL Network with posterior-mean
    parameters, extracts (mu2, sigma2, mu3, sigma3, delta1, epsilon2,
    epsilon3, psi2), joins with trial metadata, writes a CSV.

    Returns
    -------
    Path
        Path to the written CSV.
    """
```

Steps:
1. Extract posterior means for this participant from `idata.posterior`.
   Example: `omega_2_mean = float(idata.posterior.omega_2.sel(
   participant_id=participant_id).mean().values)`.
2. Build Network at the means (2-level or 3-level).
3. Convert `trials` to the binary input array: `u = np.array([t.state for
   t in trials])[:, None]` shape `(n_trials, 1)`.
4. Call `net.input_data(input_data=u, time_steps=np.ones((n_trials, 1)))`.
5. Extract per-trial attributes:
   - `mu2 = traj[BELIEF_NODE]["mean"]` -> `(n_trials,)` float
   - `sigma2 = 1.0 / traj[BELIEF_NODE]["precision"]`
   - `psi2 = traj[BELIEF_NODE]["temp"]["effective_precision"]`
   - `epsilon2 = traj[BELIEF_NODE]["temp"]["value_prediction_error"]`
   - `delta1 = traj[INPUT_NODE]["temp"]["value_prediction_error"]`
   - For 3-level only:
     - `mu3 = traj[2]["mean"]`
     - `sigma3 = 1.0 / traj[2]["precision"]`
     - `epsilon3 = traj[BELIEF_NODE]["temp"]["volatility_prediction_error"]`
6. Build a DataFrame with the schema finalized in Task 1.
7. For 2-level exports, set `mu3`, `sigma3`, `epsilon3` columns to NaN
   (dtype=float64 handles NaN cleanly).
8. Write `output_dir / f"{participant_id}_trajectories.csv"` with
   `index=False`. Return the path.

**pyhgf 0.2.10 attribute safety**: wrap every `traj[node]["temp"][key]`
read in a helper `_safe_temp(traj_node, key, shape)` that returns
`np.asarray(traj_node["temp"][key])` if present, else `np.full(shape, np.nan)`.
The research addendum verified these keys on pyhgf 0.2.10, but a defensive
fallback keeps the export robust if pyhgf's internals shift.
