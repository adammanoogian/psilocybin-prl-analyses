Create `scripts/10_plot_power_curves.py` with `from __future__ import annotations`, the usual `sys.path.insert(0, ...)` for cluster compatibility, and `matplotlib.use("Agg")` before importing pyplot.

**CLI arguments (argparse):**
- `--input-dir` (default: `RESULTS_DIR / "power"`) — directory containing `power_master.csv`, `power_a_summary.csv`, `power_b_summary.csv`
- `--precheck-dir` (default: `RESULTS_DIR / "power"`) — directory containing `trial_sweep_results.csv` from Phase 9
- `--output-dir` (default: `RESULTS_DIR / "power"`) — where to save figures
- `--bf-threshold` (default: 6.0, type float) — BF threshold for power curves (note: the roadmap specifies P(BF>6) for the figure even though the sweep used bf_threshold=10 for bf_exceeds; this script recomputes from raw bf_value column using the specified threshold)

**Define 4 plot functions (module-level, not inside main):**

1. **`plot_power_a(power_a_df: pd.DataFrame, master_df: pd.DataFrame, bf_threshold: float, save_path: Path | None) -> plt.Figure`** (VIZ-02)
   - Since `power_a_summary.csv` was computed with bf_exceeds from the sweep (threshold=10), recompute P(BF > bf_threshold) from `master_df` raw `bf_value` column:
     - Filter `master_df` to `sweep_type == "did_postdose"`
     - Add column `bf_exceeds_viz = bf_value > bf_threshold`
     - Group by `(n_per_group, effect_size)`, compute `p_bf_exceeds = mean(bf_exceeds_viz)`
   - Create figure (8x5 inches)
   - One line per unique `effect_size` value, x-axis = `n_per_group`, y-axis = `p_bf_exceeds`
   - Use distinct colors per effect size; label each line `d = {effect_size}`
   - Add horizontal dashed reference lines at y=0.80 (grey, label "80% power") and y=0.90 (light grey, label "90% power")
   - **Annotate:** Find the N where `d=0.5` crosses 0.80 (interpolate linearly between the two bracketing N values if it does not cross exactly at a grid point; if it never crosses, annotate "d=0.5 does not reach 80%"). Place a red dot and text annotation at the crossing point.
   - Axis labels: x="N per group", y="P(BF > {bf_threshold})"
   - Title: "Power Analysis A: Bayes Factor Evidence"
   - Legend outside the plot area (bbox_to_anchor) to avoid occlusion
   - Save if `save_path` is not None (PNG, 300 dpi)
   - Return figure

2. **`plot_power_b(power_b_df: pd.DataFrame, save_path: Path | None) -> plt.Figure`** (VIZ-03)
   - Create figure (8x5 inches)
   - Single line: x-axis = `n_per_group`, y-axis = `p_bms_correct`
   - Horizontal dashed reference line at y=0.75 (label "75% correct")
   - Markers at each data point
   - Axis labels: x="N per group", y="P(correct model selected)"
   - Title: "Power Analysis B: Model Discriminability"
   - Save if `save_path` is not None (PNG, 300 dpi)
   - Return figure

3. **`plot_sensitivity_heatmap(master_df: pd.DataFrame, bf_threshold: float, save_path: Path | None) -> plt.Figure`** (panel for VIZ-04)
   - Recompute P(BF > bf_threshold) from raw bf_value for `sweep_type == "did_postdose"`
   - Pivot to N x d matrix
   - Use `plt.imshow` or `sns.heatmap` (prefer matplotlib only to avoid seaborn dependency — use `ax.imshow` with `ax.set_xticks/set_yticks` and `ax.text` annotations for cell values)
   - Colormap: "RdYlGn" (red=low power, green=high power), vmin=0, vmax=1
   - Cell text: formatted as percentage (e.g., "78%")
   - Axis labels: x="Effect size (d)", y="N per group"
   - Title: "Sensitivity: P(BF > {bf_threshold})"
   - Colorbar on right
   - Save if `save_path` is not None (PNG, 300 dpi)
   - Return figure

4. **`plot_combined_figure(precheck_fig, power_a_fig, power_b_fig, heatmap_fig, save_path: Path | None) -> plt.Figure`** (VIZ-04)
   - Actually, instead of composing from existing figures (which is fragile), create a fresh 2x2 subplot figure (12x10 inches) and re-draw the four panels:
     - Panel A (top-left): Precheck recovery — read `trial_sweep_results.csv` from precheck_dir, reconstruct `SweepPoint` list, call `plot_trial_sweep` into the subplot axes. BUT since `plot_trial_sweep` creates its own figure, instead: load the CSV data and draw manually using the same style (parameter colors from `_PARAM_COLORS`, `_PARAM_MARKERS` dicts in precheck.py — import them or replicate). Draw: x=trial_count, y=r per parameter, reference line at r=0.7.
     - Panel B (top-right): Power A — same logic as `plot_power_a` but drawn into subplot axes
     - Panel C (bottom-left): Power B — same logic as `plot_power_b` but drawn into subplot axes
     - Panel D (bottom-right): Sensitivity heatmap — same logic as `plot_sensitivity_heatmap` but into subplot axes
   - Label panels "A", "B", "C", "D" in upper-left corners (bold, fontsize 14)
   - `fig.tight_layout()`
   - Save as BOTH PDF and PNG at the save_path stem (e.g., save_path="combined" -> "combined.pdf" + "combined.png"), 300 dpi
   - Return figure
