  <action>
Create `src/prl_hgf/power/curves.py` with `from __future__ import annotations` and these three public functions:

1. **`aggregate_parquets(results_dir: Path) -> pd.DataFrame`**
   - Glob for `*.parquet` files in `results_dir`
   - Read each with `pd.read_parquet(engine="pyarrow")`
   - Concatenate into a single DataFrame
   - Validate columns match `POWER_SCHEMA` keys (import from `prl_hgf.power.schema`)
   - Use `load_power_config()` to get the grid dimensions; call `total_grid_size()` with the grid lists and `n_iterations` to compute expected cell count
   - For each `sweep_type`, compute `n_expected = total_grid_size(...)` and `n_actual = len(df[df["sweep_type"] == st])`; if `n_actual < n_expected`, emit `warnings.warn(f"Missing {n_expected - n_actual} cells for sweep_type={st}: expected {n_expected}, got {n_actual}")` using the `warnings` stdlib module
   - Return the concatenated DataFrame
   - If no parquet files found, raise `FileNotFoundError` with a message including the searched directory path

2. **`compute_power_a(master_df: pd.DataFrame, bf_threshold: float = 10.0) -> pd.DataFrame`**
   - Filter to rows where `sweep_type == "did_postdose"` (primary contrast)
   - Group by `(n_per_group, effect_size)`
   - For each group, compute `p_bf_exceeds = mean(bf_exceeds)` and `n_iterations = count`
   - Return DataFrame with columns: `n_per_group`, `effect_size`, `p_bf_exceeds`, `n_iterations`
   - Also accept an optional `sweep_type: str = "did_postdose"` parameter so callers can compute curves for other contrasts too

3. **`compute_power_b(master_df: pd.DataFrame) -> pd.DataFrame`**
   - Group by `n_per_group` (aggregate across all effect sizes and sweep types, using only unique iterations — deduplicate on `(n_per_group, iteration)` since BMS values are identical across the 3 sweep_type rows per task)
   - For each group, compute `p_bms_correct = mean(bms_correct)` and `n_iterations = count`
   - Return DataFrame with columns: `n_per_group`, `p_bms_correct`, `n_iterations`

Add `__all__` with all three function names. Use NumPy-style docstrings for all functions. Import `from prl_hgf.power.schema import POWER_SCHEMA` and `from prl_hgf.power.grid import total_grid_size` and `from prl_hgf.power.config import load_power_config`.

Update `src/prl_hgf/power/__init__.py` to import and re-export `aggregate_parquets`, `compute_power_a`, `compute_power_b` from `curves`.
