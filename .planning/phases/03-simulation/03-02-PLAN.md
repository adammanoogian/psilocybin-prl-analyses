30 participants x 3 sessions = 180 participant-sessions. Each participant-session
produces a trial-level DataFrame row. The output is a single tidy CSV with
ground-truth parameters embedded as columns.

recovery). The group x session structure must correctly encode the hypothesized
differences between post_concussion and healthy_control groups, including session
deltas that simulate treatment effects.

`scripts/03_simulate_participants.py` pipeline script, and batch-level unit tests.

**`batch.py`** — single public function `simulate_batch`:

```
def simulate_batch(
    config: AnalysisConfig,
    output_path: Path | None = None,
) -> pd.DataFrame:
```

Implementation:

1. **Seed strategy:** Derive per-(participant, session) seeds from `config.simulation.master_seed`. Create a master RNG: `rng_master = np.random.default_rng(config.simulation.master_seed)`. Draw all seeds upfront: `seeds = rng_master.integers(0, 2**31, size=(n_total, 2))` where `n_total = n_groups * n_per_group * n_sessions`. `seeds[i, 0]` is the env seed (for `generate_session`), `seeds[i, 1]` is the sim seed (for `simulate_agent` choices). Use a flat index `i` that increments as you iterate over groups, participants, sessions.

2. **Session labels:** Read from `config.simulation.session_deltas[group_name].session_labels` and prepend "baseline". So session_labels = ["baseline", "post_dose", "followup"], session_idx 0/1/2.

3. **JIT pre-warm:** Before the batch loop, build one dummy 3-level network and call `input_data` once with dummy data to trigger JAX JIT compilation. This reduces first-participant latency.

4. **Main loop:** Iterate over groups (sorted keys of `config.simulation.groups`), then participants (0 to n_per_group-1), then sessions (0 to 2). For each:
   a. Get `group_cfg = config.simulation.groups[group_name]` and `session_cfg = config.simulation.session_deltas[group_name]`.
   b. Create sim RNG: `rng_sim = np.random.default_rng(seeds[flat_idx, 1])`.
   c. Sample params via `sample_participant_params(group_cfg, session_cfg, session_idx, rng_sim)`.
   d. Build fresh network: `net = build_3level_network(omega_2=params["omega_2"], omega_3=params["omega_3"], kappa=params["kappa"])`.
   e. Generate trial sequence: `trials = generate_session(config, seeds[flat_idx, 0])`.
   f. Simulate: `result = simulate_agent(net, trials, params["beta"], params["zeta"], rng_sim)`.
   g. Build rows: one row per trial, with columns matching SIM-06:
      - `participant_id`: `f"{group_name}_{participant_idx:03d}"` (e.g., "post_concussion_000")
      - `group`: group_name
      - `session`: session_label (e.g., "baseline")
      - `session_idx`: 0, 1, or 2
      - `trial`: trial.trial_idx
      - `cue_chosen`: result.choices[t]
      - `reward`: result.rewards[t]
      - `cue_0_prob`, `cue_1_prob`, `cue_2_prob`: trial.cue_probs[0/1/2]
      - `phase_label`: trial.phase_label
      - `phase_name`: trial.phase_name
      - `best_cue`: trial.best_cue
      - `true_omega_2`, `true_omega_3`, `true_kappa`, `true_beta`, `true_zeta`: from params dict
      - `model`: "hgf_3level" (always, per research recommendation)
   h. Increment flat_idx.

5. **Progress logging:** Print progress every 10 participant-sessions: `f"[{flat_idx}/{n_total}] {group_name} P{participant_idx:03d} session {session_label} done ({elapsed:.1f}s)"`. Use `time.time()` for elapsed.

6. **Output:** Concatenate all rows into a single DataFrame. If `output_path` is provided, save as CSV via `df.to_csv(output_path, index=False)` and print the path. Return the DataFrame.
