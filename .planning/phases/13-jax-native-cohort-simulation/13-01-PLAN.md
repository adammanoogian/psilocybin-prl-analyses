**Architecture: factory pattern (required for vmap compatibility).**

Building a pyhgf `Network()` involves Python-side side effects that cannot be
traced by JAX. If `simulate_session_jax` builds the network internally, then
`jax.vmap(simulate_session_jax, ...)` will fail during tracing. The solution
is the same pattern used by `build_logp_ops_batched` in `hierarchical.py`:
build the network ONCE at factory time, then pass the captured `scan_fn` and
`base_attrs` into a pure-JAX function that can be vmapped.

**Three functions:**

1. `_build_session_scanner() -> tuple[callable, dict]` — builds pyhgf Network
   once, returns `(scan_fn, base_attrs)`. NOT traceable by JAX. Called once
   outside vmap/jit.
2. `_run_session(scan_fn, base_attrs, omega_2, omega_3, kappa, beta, zeta,
   cue_probs_arr, rng_key) -> (choices, rewards, diverged)` — pure JAX function,
   vmappable. Contains parameter injection, lax.scan, clamping.
3. `simulate_session_jax(omega_2, omega_3, kappa, beta, zeta, cue_probs_arr,
   rng_key) -> (choices, rewards, diverged)` — thin public wrapper: calls
   `_build_session_scanner()` then `_run_session(...)`.

**`_build_session_scanner` function:**
```python
def _build_session_scanner() -> tuple[callable, dict]:
    """Build scan_fn and base_attrs once (not traceable by JAX)."""
    net = build_3level_network()
    net.input_data(
        input_data=np.zeros((1, 3)),
        observed=np.ones((1, 3), dtype=int),
    )
    return net.scan_fn, net.attributes
```

**`_run_session` function signature:**
```python
def _run_session(
    scan_fn,       # captured from _build_session_scanner
    base_attrs,    # captured from _build_session_scanner
    omega_2: jnp.ndarray,
    omega_3: jnp.ndarray,
    kappa: jnp.ndarray,
    beta: jnp.ndarray,
    zeta: jnp.ndarray,
    cue_probs_arr: jnp.ndarray,  # shape (n_trials, 3)
    rng_key: jnp.ndarray,
) -> tuple[jnp.ndarray, jnp.ndarray, jnp.ndarray]:
```

Returns `(choices, rewards, diverged)` where:
- `choices`: shape `(n_trials,)`, int32 — chosen cue per trial
- `rewards`: shape `(n_trials,)`, int32 — binary reward per trial
- `diverged`: scalar bool — True if any trial was clamped

**`simulate_session_jax` function (thin public wrapper):**
```python
def simulate_session_jax(
    omega_2: jnp.ndarray,
    omega_3: jnp.ndarray,
    kappa: jnp.ndarray,
    beta: jnp.ndarray,
    zeta: jnp.ndarray,
    cue_probs_arr: jnp.ndarray,  # shape (n_trials, 3)
    rng_key: jnp.ndarray,
) -> tuple[jnp.ndarray, jnp.ndarray, jnp.ndarray]:
    scan_fn, base_attrs = _build_session_scanner()
    return _run_session(
        scan_fn, base_attrs,
        omega_2, omega_3, kappa, beta, zeta,
        cue_probs_arr, rng_key,
    )
```

**`_run_session` implementation pattern (follow 13-RESEARCH.md exactly):**

1. **Parameter injection** (copy exact pattern from hierarchical.py `_single_logp_3level` lines 404-425):
   - Shallow-copy `attrs = dict(base_attrs)`
   - omega_2 into nodes 1, 3, 5 `tonic_volatility`
   - omega_3 into node 6 `tonic_volatility`
   - kappa into node 6 `volatility_coupling_children = jnp.array([kappa, kappa, kappa])`
   - kappa into nodes 1, 3, 5 `volatility_coupling_parents = jnp.array([kappa])`

2. **Define `_sim_step(carry, x)` inner function:**
   - `carry = (attrs, rng_key, prev_choice)` — note: `beta` and `zeta` come from outer closure
   - `x = cue_probs` — shape `(3,)` float, sliced by lax.scan from `cue_probs_arr`

   Step body:
   a. **Read prior beliefs** from carry attrs BEFORE scan_fn call:
      `p_reward = jnp.array([attrs[0]["expected_mean"], attrs[2]["expected_mean"], attrs[4]["expected_mean"]])`
   b. **Split PRNG key**: `step_key, next_key = jax.random.split(rng_key)` then
      `choice_key, reward_key = jax.random.split(step_key)`
   c. **Softmax-stickiness choice**: `stick = (prev_choice == jnp.arange(3)).astype(jnp.float32)`,
      `logits = beta * p_reward + zeta * stick`,
      `choice = jax.random.categorical(choice_key, logits)`
   d. **Bernoulli reward**: `reward = jax.random.bernoulli(reward_key, cue_probs[choice])`,
      `reward = jnp.int32(reward)`
   e. **Build per-trial scan input for scan_fn** (must match pyhgf expected structure):
      ```
      reward_f = jnp.float32(reward)
      values_t = (
          jnp.where(choice == 0, reward_f, jnp.float32(0.0)).reshape(1),
          jnp.where(choice == 1, reward_f, jnp.float32(0.0)).reshape(1),
          jnp.where(choice == 2, reward_f, jnp.float32(0.0)).reshape(1),
      )
      observed_t = (
          jnp.where(choice == 0, jnp.float32(1.0), jnp.float32(0.0)),
          jnp.where(choice == 1, jnp.float32(1.0), jnp.float32(0.0)),
          jnp.where(choice == 2, jnp.float32(1.0), jnp.float32(0.0)),
      )
      time_step_t = jnp.float32(1.0)
      scan_input_t = (values_t, observed_t, time_step_t, None)
      ```
      **Shape contract (verified from hierarchical.py `_build_scan_inputs`):**
      - `values_t` elements: shape `(1,)`. In the logp path, `input_data[:, 0:1]`
        produces shape `(n_trials, 1)`, which `lax.scan` slices to `(1,)` per step.
        Use `.reshape(1)` unconditionally.
      - `observed_t` elements: scalar. In the logp path, `observed[:, 0]` produces
        shape `(n_trials,)`, which `lax.scan` slices to a scalar per step.
      - `time_step_t`: scalar `jnp.float32(1.0)`.
      - Fourth element: `None` (no missing-data mask in simulation).
      This is the single correct format. Do NOT add fallback branches.
   f. **HGF update**: `new_attrs, _ = scan_fn(attrs, scan_input_t)`
   g. **Layer 2 clamping** (exact copy from hierarchical.py `_clamped_step`):
      ```
      leaves = jax.tree_util.tree_leaves(new_attrs)
      all_finite = jnp.all(jnp.array([jnp.all(jnp.isfinite(leaf)) for leaf in leaves]))
      mu_2_vals = jnp.array([new_attrs[1]["mean"], new_attrs[3]["mean"], new_attrs[5]["mean"]])
      mu_2_ok = jnp.all(jnp.abs(mu_2_vals) < _MU_2_BOUND)
      is_stable = all_finite & mu_2_ok
      safe_attrs = jax.tree_util.tree_map(lambda n, o: jnp.where(is_stable, n, o), new_attrs, attrs)
      ```
   h. **Return**: `(safe_attrs, next_key, choice), (choice, reward, is_stable)`

4. **Initial carry**: `(attrs, rng_key, jnp.int32(-1))` — `-1` sentinel for no stickiness on trial 0

5. **Run lax.scan**: `_, (choices, rewards, stability_flags) = lax.scan(_sim_step, init_carry, cue_probs_arr)`

6. **Compute diverged**: `diverged = jnp.any(~stability_flags)`

7. **Return**: `(choices, rewards, diverged)`

**Docstrings (NumPy-style for all three functions):**

- `_run_session`: Document that this is the pure-JAX simulation kernel, designed to be

- `simulate_session_jax`: Document as the public convenience wrapper. Notes that for

**IMPORTANT anti-patterns to avoid:**
