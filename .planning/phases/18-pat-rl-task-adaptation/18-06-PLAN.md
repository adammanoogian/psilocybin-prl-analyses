Shape:

```python
"""PAT-RL foundation smoke: simulate -> fit -> export 5 agents on CPU.

Runs the full parallel PAT-RL stack end-to-end to verify all Phase 18 modules
compose correctly.  Writes trajectory CSVs + parameter summary CSV under
``output/patrl_smoke/`` (or the dir passed via ``--output-dir``).

Usage
-----
    python scripts/12_smoke_patrl_foundation.py [--level {2,3}] [--output-dir DIR]
    [--n-tune N] [--n-draws N] [--seed N]

Exits 0 on success; exits non-zero (and prints diagnostic) if any step fails
or any output CSV is missing / malformed.
"""

from __future__ import annotations

import argparse
import logging
import sys
from pathlib import Path

import numpy as np
import pandas as pd

from config import OUTPUT_DIR
from prl_hgf.env.pat_rl_config import load_pat_rl_config
from prl_hgf.env.pat_rl_sequence import generate_session_patrl
from prl_hgf.fitting.hierarchical_patrl import fit_batch_hierarchical_patrl
from prl_hgf.analysis.export_trajectories import (
    export_subject_trajectories, export_subject_parameters,
)
from prl_hgf.models.hgf_2level_patrl import build_2level_network_patrl
from prl_hgf.models.hgf_3level_patrl import build_3level_network_patrl
from prl_hgf.models.response_patrl import model_a_logp, expected_value
```

Main steps (implement as separate helper functions for clarity):

1. `_parse_args()` — argparse with `--level`, `--output-dir`, `--n-tune`,
   `--n-draws`, `--seed`, `--n-participants`.

2. `_simulate_cohort(n_participants, level, master_seed, config)`:
   - For each participant `i`: sample true parameters from
     `config.simulation.phenotypes["healthy"]` (use
     `rng.normal(mean, sd)` for each phenotype param).
   - `trials = generate_session_patrl(config, seed=child_seed)`.
   - Run a forward pass through the HGF at the true params to get `mu2`
     per trial; sample a choice per trial via softmax-EV with the true
     beta (use `np.random.default_rng(...).choice` on 2 options weighted
     by `exp(beta*EV)`).
   - Draw the outcome per trial conditioned on `(state, choice)` using
     `config.task.contingencies`.
   - Build a sim_df row per trial with the full column set (including
     the true parameters in a companion dict for recovery sanity check).
   - Return `sim_df`, `true_params_by_participant` dict, `trials_by_participant` dict.

3. `_fit(sim_df, level, n_tune, n_draws, seed)`:
   - `model_name = "hgf_2level_patrl" if level == 2 else "hgf_3level_patrl"`.
   - Call `fit_batch_hierarchical_patrl(sim_df, model_name=model_name,
     response_model="model_a", n_chains=2, n_tune=n_tune, n_draws=n_draws,
     random_seed=seed)`.
   - Return the InferenceData.

4. `_export(idata, trials_by_participant, choices_by_participant, level, output_dir)`:
   - For each participant: call `export_subject_trajectories(...)`.
   - Call `export_subject_parameters(idata, model_name, output_dir)`.
   - Return list of paths written.

5. `_sanity_check(idata, true_params, tol_bpm=None)`:
   - Log per-participant: posterior mean omega_2 vs true, posterior mean
     beta vs true. NOT a hard assertion — just a logged diff per
     parameter per participant.
   - HARD assertions:
     - All posterior means finite.
     - `idata.sample_stats.diverging.sum() / idata.sample_stats.diverging.size < 0.20`
       (<20% divergence rate is the relaxed smoke gate).
   - Soft (log-only) check: average across participants,
     `sign(posterior_mean_omega_2 - prior_mean_omega_2)` matches the sign
     of `(true_omega_2 - prior_mean_omega_2)` — the fit moved in the
     right direction. Log a warning if not, but don't exit non-zero
     (this is a smoke, not a recovery gate).

6. `main()`:
   - Parse args.
   - Set up logging via `logging.basicConfig(...)` at INFO.
   - Print a header banner stating: `"PAT-RL foundation smoke — Phase 18"`.
   - Load config via `load_pat_rl_config()`.
   - Create `output_dir = OUTPUT_DIR / "patrl_smoke"` if not passed;
     `output_dir.mkdir(parents=True, exist_ok=True)`.
   - Run simulate → fit → export → sanity_check.
   - Print final summary: `"WROTE N CSVs at output/patrl_smoke"`.
   - Exit 0.
   - Any exception: log it with `logger.exception(...)`, exit 1.

Smoke budget: target <10 minutes total on a laptop CPU with
`--n-participants 5 --n-tune 200 --n-draws 200`.

**NOTE**: Python 3.10+ `__future__ annotations` at top; NumPy-style
docstrings; absolute imports only.
