<objective>
Implement the core Phase 19 Laplace fit pipeline. The function
`fit_vb_laplace_patrl` runs quasi-Newton MAP on `_build_patrl_log_posterior`
(imported unmodified from `hierarchical_patrl.py`), takes the autodiff
Hessian at the mode, PD-regularizes if indefinite via eigh-clip, and
packages the result as an `az.InferenceData` via
`build_idata_from_laplace` from Plan 19-02 — producing a fit result that
Plan 18-05's exporters consume unchanged.

Key architectural decisions (from research):
- Use `jaxopt.LBFGS` on the dict-of-arrays PyTree directly (jaxopt is
  installed + works PyTree-native; migrating to `optax.lbfgs` is a
  future escape hatch if jaxopt stops working with a jax upgrade).
- Use `jax.flatten_util.ravel_pytree` to bridge from dict to flat vector
  for `jax.hessian` — `ravel_pytree` flattens in dict-insertion order so
  we define canonical parameter orders (imported from Plan 19-02's
  module).
- Block-diagonal Hessian property is documented in a TODO comment, but
  Phase 19 computes the DENSE `(P*K) × (P*K)` Hessian (tractable for
  P≤200, K∈{2,5}) for implementation simplicity. Block-structured
  computation is a Phase 20+ optimization.
- Eigh-clip PD regularization (from research Q4): `eigh -> clip
  eigenvalues at eps=1e-8 -> reconstruct`. Cleaner than ridge loop;
  deterministic; logs diagnostics.
- Kappa is fit in native space (not logit-reparametrized), matching the
  existing TruncatedNormal prior in `_build_patrl_log_posterior`. Init
  is clipped into `(lower+1e-6, upper-1e-6)`. Logit reparam is OQ2,
  deferred to a follow-up if cluster smoke shows kappa at boundary.

def fit_vb_laplace_patrl(
    sim_df: pd.DataFrame,
    model_name: str,
    response_model: str = "model_a",
    config: PATRLConfig | None = None,
    n_pseudo_draws: int = 1000,
    max_iter: int = 200,
    tol: float = 1e-5,
    n_restarts: int = 1,
    random_seed: int = 0,
) -> az.InferenceData:
    """Fit PAT-RL via Laplace approximation at the MAP.

    Parameters
    ----------
    sim_df : pd.DataFrame
        Simulated or observed data matching the hierarchical_patrl input
        contract (see ``_build_arrays_single_patrl`` docstring for
        required columns).
    model_name : str
        ``'hgf_2level_patrl'`` or ``'hgf_3level_patrl'``.
    response_model : str, default 'model_a'
        Only ``'model_a'`` is supported in Phase 19. Other values raise
        NotImplementedError (Phase 20+ scope).
    config : PATRLConfig, optional
        Analysis config. Defaults to ``load_pat_rl_config()``.
    n_pseudo_draws : int, default 1000
        Number of pseudo-samples from N(mode, Sigma). See
        ``build_idata_from_laplace`` for note on why this does not affect
        Laplace accuracy.
    max_iter : int, default 200
        jaxopt.LBFGS maximum iterations.
    tol : float, default 1e-5
        jaxopt.LBFGS convergence tolerance.
    n_restarts : int, default 1
        Number of MAP optimization restarts from perturbed initial
        positions. If >1, the restart with highest log-posterior at the
        mode is kept. Default 1 = single run from prior means.
    random_seed : int, default 0
        Seed for (a) MultivariateNormal pseudo-samples, (b) any
        restart-perturbation RNG.

    Returns
    -------
    az.InferenceData
        Posterior group with vars matching NUTS output (``omega_2,
        log_beta, beta`` for 2-level; +``omega_3, kappa, mu3_0`` for
        3-level) shaped ``(chain=1, draw=n_pseudo_draws,
        participant_id=P)``. ``sample_stats`` group includes diagnostic
        scalars (converged, n_iterations, logp_at_mode,
        hessian_min_eigval, ridge_added).
    """
```

   def _regularize_to_pd(H: np.ndarray, eps: float = 1e-8
   ) -> tuple[np.ndarray, dict[str, float]]:
       """Clip negative/tiny eigenvalues to eps; return (H_pd, diagnostics).

       Returned diag dict uses the canonical Phase 19 key set:
       ``hessian_min_eigval``, ``hessian_max_eigval``,
       ``n_eigenvalues_clipped``, ``ridge_added``. Plan 19-02 consumes
       these names directly.
       """
       w, V = np.linalg.eigh(H)
       n_clipped = int(np.sum(w < eps))
       w_clip = np.maximum(w, eps)
       H_pd = V @ np.diag(w_clip) @ V.T
       diag = {
           "hessian_min_eigval": float(np.min(w)),
           "hessian_max_eigval": float(np.max(w)),
           "n_eigenvalues_clipped": float(n_clipped),
           "ridge_added": float(max(0.0, eps - float(np.min(w)))),
       }
       return H_pd, diag
   ```
